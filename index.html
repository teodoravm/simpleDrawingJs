You have two containers that can hold tile objects in them. The containers have a width of 500 pixels and a minimum height of 400 pixels. The tiles have a constant size of 120x40 pixels.

  1. Implement adding of new tiles on random position in the left container. The tiles should use the text from the input field below the containers. Put your implementation in the addTile() method.
  2. Sort and position the tiles in the left container so that they don't overlap. The tiles should be sorted by name and positioned using their "left" and "top" properties. Put your implementation in the sortTiles() method.
  3. Extend the sorting by adding space of 10 pixels between tiles.
  4. Implement moving between containers by tile click.
  5. Extend the moving by maintaining the sorted order of the tiles in both containers after tile is moved.
  6. Implement Drag and Drop between containers.
